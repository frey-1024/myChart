# myChart
myChart = webpack2 + react + router + redux + scss + nodejs + mysql + es6/7; 
<br/>
功能介绍：http://www.cnblogs.com/zhangkunweb/p/6853728.html
<br/>
<br/>
(1) api 的使用<br/>
    cd api<br/>
    npm install<br/>
    npm start<br/>
(2) web 的使用<br/>
    cd web<br/>
    npm istall<br/>
  
  热加载：<br/>
    npm run hot <br/>
  发布：<br/>
    npm run dist<br/>
